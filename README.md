# API de Filmes

## Como executar o projeto

1. Instale as dependências:

```
npm install
```

2. Inicie o servidor:

```
node server.js
```

3. A API estará disponível em:

```
http://localhost:8080
```

---

## Rota disponível

### GET /api/filmes

Retorna a lista de filmes cadastrados.

---

## Exemplo de uso

Acesse no navegador ou no Postman:

```
http://localhost:8080/api/filmes
```
